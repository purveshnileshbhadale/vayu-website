About

What is VAYU?

VAYU is a next-generation AI voice assistant that runs natively on your desktop. Unlike cloud-only assistants, VAYU integrates directly with your operating system — controlling applications, managing files, browsing the web, and executing code — all through natural voice commands.

Built with a modular plugin architecture and multi-provider AI fallback (Gemini → OpenRouter → Groq), VAYU ensures you always have a responsive, intelligent assistant at your command — even if one AI provider goes down.

Created by Purvesh Nilesh Bhadale — independently developed, maintained, and freely available for everyone.

Get Started

Download VAYU

Choose your platform. VAYU is completely free — no subscriptions, no hidden costs.

🪟

Windows

Standalone EXE for Windows 10 & 11 (64-bit)

Download for Windows ~27 MB

Requirements

Fully standalone — no Python or dependencies needed. All libraries are compiled inside. Just download and run.

Setup Steps

1
Download VAYU.exe

Click the "Download for Windows" button above to get the standalone EXE.

2
Run VAYU

Double-click VAYU.exe — no Python or dependencies needed. On first run, you'll be prompted to enter your API keys.

config/api_keys.json
{
"gemini_api_key": "YOUR_GEMINI_KEY",
"openrouter_api_key": "YOUR_OPENROUTER_KEY",
"groq_api_key": "YOUR_GROQ_KEY",
"os_system": "windows"
}
3
Add API Keys

On first launch, VAYU shows a Settings dialog. Paste your Gemini and OpenRouter API keys there, then click Save.

🍎

Setup Steps

1
Download & Extract

Click "Download for macOS" above to get the ZIP. Extract it anywhere on your Mac.

2
Run VAYU

Double-click start_vayu.command — it auto-installs Python dependencies and launches VAYU. On first run, you'll be prompted to enter your API keys.

If you get a security warning, right-click the file and select Open.

$ pip install -r requirements.txt
5
Run VAYU

Double-click VAYU.command in the extracted folder → Terminal opens and VAYU starts automatically.

Or via Terminal
$ python main.py
6
Enter API Keys in the UI

On first launch, VAYU shows a Settings dialog. Paste your Gemini, OpenRouter, and Groq API keys there → click Save. No manual file editing needed.
